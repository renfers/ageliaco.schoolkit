ageliaco.schoolkit Installation
-------------------------------

To install ageliaco.schoolkit using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ageliaco.schoolkit`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ageliaco.schoolkit
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
